## はじめに

### 素人でも戦略的に取引したい

株やFXの知識がないため，戦略的な取引ができない素人が多いと思います．私もその一人です．
そこで，強化学習の力を借りて戦略的に取引をしようと思います．



### AIncomeプロジェクト

AIで所得（Income）を得るため「AIncome」と名付けました．
